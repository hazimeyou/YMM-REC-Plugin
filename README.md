# YMM-REC-Plugin

YukkuriMovieMaker4（YMM4）向けの**マイク録音プラグイン**です。

ffmpeg を用いた音声録音と、NAudio によるリアルタイム音量メーター表示を提供します。

---

## 主な機能

* 🎙 マイク音声の録音
* 📊 リアルタイム音量レベルメーター表示
* 🎚 録音音量（ゲイン）調整
* 🔽 使用するマイクデバイスの選択
* ⏱ 日時ベースの自動ファイル名生成

---

## 動作環境

* Windows 11
* YukkuriMovieMaker 4

---

## 使用ライブラリ

* NAudio
* ffmpeg
---

## セットアップ

1. YukkuriMovieMaker 4 を公式リリースページからダウンロード
2. YukkuriMovieMaker 4 を起動
3. プラグインを導入
4. ツールバーから **録音プラグイン** を起動

※ ffmpeg はプラグイン側で自動的に処理されます。

---

## 使い方

1. マイクを選択
2. 録音音量を調整
3. 「録音開始」を押す
4. 「録音停止」で WAV ファイルが保存されます

保存先はプラグイン用ディレクトリです。

---

## 注意事項

* 録音停止後、保存処理のため少し待つ場合があります
* 音量メーターは**視覚確認用**です
